# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
docker compose up -d
```

Заполняем mongodb данными

```shell
chmod +x mongo-init.sh
./mongo-init.sh
```

## Как проверить

```shell
curl "http://localhost:8080/helloDoc/count"
```

## Как остановить
```shell
docker-compose down -v
```
